# newsh - a bash script generator
<img width="698" height="743" alt="image" src="https://github.com/user-attachments/assets/941ed656-eca3-475e-869c-6d387603295d" />

This tool generates a new script with my preferred settings: strict mode and extglob enabled.

This was made for bash scripting but it also accepts a template to use in place of the original one. 

This makes it compatible with any shell and any user preference.
